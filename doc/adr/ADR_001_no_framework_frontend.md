# ADR 001 — Arquitetura frontend sem framework

## Status
Aceito

## Contexto
Decisão de não utilizar framework frontend para manter simplicidade, controle total
e compatibilidade com GitHub Pages.

## Decisão
Utilizar HTML, CSS e JavaScript puro com ES Modules.

## Consequências
Código explícito, menos dependências e maior disciplina arquitetural.
